Summary

Designed and implemented a 32-bit RISC-V CPU using digital circuit design in Logisim. The processor executes RISC-V assembly instructions through carefully designed hardware components including datapath elements, control logic, and memory interfaces.

This project provided hands-on experience with computer architecture fundamentals, demonstrating how software instructions translate to hardware execution. The modular design approach enabled systematic development and testing of individual components before system integration.
