RISC-V CPU Design

Completed

Overview

Designed and implemented a 32-bit RISC-V CPU from scratch as part of UC Berkeley's CS61C (Computer Architecture) course. Built a fully functional processor capable of executing RISC-V assembly instructions using digital circuit design in Logisim.

Key Components

The implementation involved designing core CPU components including datapath elements, control logic, and memory interfaces. The processor successfully executes RISC-V instructions through carefully designed hardware circuits.

Technologies & Tools

Logisim RISC-V ISA Digital Logic Hardware Design Assembly Language

What I Learned

This project provided hands-on experience with low-level computer architecture concepts, including instruction set architectures, datapath design, and control logic. It deepened my understanding of how high-level code translates to machine instructions and executes on physical hardware.
